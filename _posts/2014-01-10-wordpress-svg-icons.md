<a href="http://wordpress.org/plugins/svg-vector-icon-plugin/" target="_blank">WordPress Icons &#8211; SVG</a> es un plugin para WordPress que contiene mas de 500 iconos en formato **SVG**. El plugin ha sido creado por <a href="http://www.evan-herman.com/" target="_blank">Evan Herman</a>.

<!--more-->

Una vez que tengamos instalado el plugin su uso es bastante sencillo. Lo primero es localizar el icono que queremos utilizar entre la lista de todos los que tenemos a nuestra disposición, una vez localizado hacemos clic en el icono y el código del icono aparecerá en la parte superior.

![ejemplo svg wordpress][2]

Ahora copiamos el código, lo pegamos en el post, sección o lugar de wordpress donde queremos que vaya y podemos ver el resultado justo debajo.


{% highlight HTML linenos %}
<span data-icon="&#xe005;"></span>
{% endhighlight %}

Podemos ampliar el tamaño del icono añadiendo un `style="font-size:3em;"`

También podemos cambiar el color del icono añadiendo un `style="color:red;"`


Todas estas operaciones y muchas mas las podemos llevar a cabo desde una demo que esta disponible en la web de Evan Hermann la cual simplifica bastante el trabajo y nos ayuda a ver al instante los cambios que vamos haciendo en los iconos. **Un trabajo impresionante.**
